✨ Mixorix 🧱🤖 — Foundations Backed by Intelligence

An intelligent concrete mix design assistant for civil engineers and construction professionals. Mixorix uses AI to generate optimal mix designs based on grade, slump, exposure, and workability requirements. It delivers detailed proportions, visual charts, and smart suggestions – all tailored to IS code standards.

  • ✨ Features (planned): AI-powered mix ratio generation, IS 10262:2019 compliance, workability & W/C optimization, strength prediction, and PDF report generation with visual pie/bar charts.
  • Frontend: React.js, Tailwind CSS, Recharts, Framer Motion, React Router v6, Context API
  • Backend: Node.js, Express, MongoDB, Flask (Python) for AI model integration
  • AI Models: Rule-based logic for mix design, ML for strength prediction (regression), future scope for cost optimization & site-based AI suggestions
  • Status: 🚧 In Progress – Mix generation module complete, working on charts and UI polish
  • Repo: GitHub | Live: Coming Soon
